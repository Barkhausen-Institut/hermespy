scenario package¶

scenario.scenario module¶

class Scenario(parameters=None, param_general=None, rnd=None)¶

Bases: object

Implements the simulation scenario.

The scenario contains objects for all the different elements in a given simulation, such as modems, channel models, bit sources, etc.

tx_modems¶

list of all modems to be used for transmission with ‘number_of_tx_modems’ elements.

Type

list(Modem)

rx_modems¶

list of all modems to be used for reception with ‘number_of_rx_modems’ elements.

Type

list(Modem)

sources¶

list of all bits sources. Each bit source will be associated to one transmit modem.

Type

list(BitsSource)

rx_samplers¶

list of all receive re-samplers. Each resampler will be associated to one receive modem

Type

list(RxSampler)

channels¶

list of all propagation channels. channels[i][j] contains the channel from transmit modem ‘i’ to receive modem ‘j’

Type

list(list(Channel))

noise¶

list of all noise sources. Each noise source is associated to one receive modem

Type

list(Noise)

init_drop()¶

Intializes variables for each drop or creates new random numbers.

Return type

None

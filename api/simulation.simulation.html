propagate(transmitted_signals, *, _ray_trace_ctx=None)

Propagate the signals generated by registered transmitters over the channel model.

Signals receiving at each receive modem are a superposition of all transmit signals impinging onto the receive modem over activated channels.

The signal stream matrices contain the number of antennas on the first dimension and the number of signal samples on the second dimension

Parameters

transmitted_signals (List[List[Optional[np.ndarray]]]) – Signal models transmitted by each registered device.

Returns

A square matrix of dimension num_devices containing tuples of propagated signals as well as the respective channel state information.

Return type

np.ndarray

Raises

ValueError – If the number of transmitted_signals does not equal the number of devices.

receive_operators(drop_run_flag=None, *, _ray_trace_ctx=None)

Generate signals received by all receivers registered with this scenario.

This routine superimposes propagated signals at the receiver-inputs, introduces additive thermal noise and mixes according to carrier frequency configurations.

Parameters

drop_run_flag (np.ndarray, optional) – Mask that says if signals are to be created for specific snr.

Returns

A list of M tuples containing the received noisy base-band signals, the channel impulse response as well as the noise variance for each receiving modem, respectively.

Return type

List[Tuple[Signal, ChannelStateInformation, float]]

Raises

ValueError – If the number of arriving_signals does not equal the number of registered receive modems.

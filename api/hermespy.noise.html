class Noise(random_generator=None)

Bases: object

Implements a complex additive white gaussian noise at the receiver.

__receiver

Receiver modem this noise model is attached to. None if the noise model is currently floating.

Type

Optional[Receiver]

__random_generator

Random number generator. If not set (i.e. None), the random generator instance of the attached receiver will be called.

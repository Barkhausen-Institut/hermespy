class PowerAmplifier(model=Model.NONE, tx_power=1.0, saturation_amplitude=0.0, input_backoff_pa_db=0.0, rapp_smoothness_factor=1.0, saleh_alpha_a=2.0, saleh_alpha_phi=1.0, saleh_beta_a=2.0, saleh_beta_phi=1.0, custom_pa_input=None, custom_pa_output=None, custom_pa_gain=None, custom_pa_phase=None, adjust_power_after_pa=False)

Bases: object

This module implements a distortion model for a power amplifier (PA).

The following PA models are currently supported:

  • Rapp’s model, as described in C. Rapp, “Effects of HPA Nonlinearity on a 4-PI DPSK/OFDM Signal for a digital Sound Broadcasting System” in Proc. Eur. Conf. Satellite Comm., 1991

  • Saleh’s model, as described in A.A.M. Saleh, “Frequency-independent and frequency dependent nonlinear models of TWT amplifiers” in IEEE Trans. Comm., Nov. 1981

  • any custom AM/AM and AM/PM response as a vector

Currently only memoryless models are supported.

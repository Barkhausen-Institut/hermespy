class PseudoRandomGenerator(init_sequence, offset=1600)

Bases: object

A rng for pseudo-random bit sequences.

See also TS 138 211 section 5.2.1 for details.

Class initialization.

Generators with identical initialization will output identical random sequences!

Parameters

  • init_sequence (np.ndarray) – A sequence of 31 bits initializing the rng.

  • offset (int) – Gold sequence parameter controlling the sequence offset.

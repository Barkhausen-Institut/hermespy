===============
Simulated Drop
===============

.. inheritance-diagram:: hermespy.simulation.drop.SimulatedDrop
   :parts: 1

Simulated drops are the result of a single MonteCarlo simulation iteration within HermesPy.
They contain all of the information generated by simulated devices as well as their estimated receptions.

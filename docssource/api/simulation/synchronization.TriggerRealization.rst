====================
Trigger Realization
====================

.. inheritance-diagram:: hermespy.simulation.simulated_device.TriggerRealization
   :parts: 1

:class:`TriggerRealizations<hermespy.simulation.simulated_device.TriggerRealization>` are
generated once per :class:`SimulatedDrop<hermespy.simulation.drop.SimulatedDrop>` for each unique
:class:`TriggerModel<hermespy.simulation.simulated_device.TriggerModel>` and contain information
about the synchronized simulated devices' assumed time offsets.

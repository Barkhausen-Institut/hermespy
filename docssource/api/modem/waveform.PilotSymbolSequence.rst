=====================
Pilot Symbol Sequence
=====================

.. inheritance-diagram:: hermespy.modem.waveform.PilotSymbolSequence
   :parts: 1

This class provides a framework for defining and managing pilot symbol sequences, which are used in various modulation schemes to assist in channel estimation and synchronization.
It serves as a base class and can be extended to implement specific pilot symbol patterns for different communication systems.
